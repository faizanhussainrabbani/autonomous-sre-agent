# Code Of Conduct

This project is committed to providing a welcoming, safe, and inclusive
environment for everyone.

## Our Standards

Examples of behavior that helps create a positive community:

- Be respectful, kind, and constructive.
- Assume good intent and give actionable feedback.
- Focus on technical outcomes, not personal attacks.
- Be open to different viewpoints and experiences.
- Accept responsibility and correct mistakes quickly.

Examples of unacceptable behavior:

- Harassment, threats, or intimidation.
- Personal attacks, insults, or discriminatory language.
- Trolling, deliberate disruption, or bad-faith engagement.
- Publishing private or sensitive information without consent.
- Any conduct that is inappropriate in a professional setting.

## Scope

This Code of Conduct applies in all project spaces, including issues, pull
requests, discussions, and other community channels.

## Reporting

If you experience or witness behavior that violates this Code of Conduct,
report it to the maintainers by opening a private report in GitHub Security
Advisories when possible, or by contacting @faizanhussain directly on GitHub.

## Enforcement

Maintainers are responsible for clarifying standards and taking fair,
consistent action in response to unacceptable behavior. They may remove,
edit, or reject comments, commits, code, issues, and other contributions that
violate this Code of Conduct.

Repeated or severe violations may result in temporary or permanent bans from
project participation.
